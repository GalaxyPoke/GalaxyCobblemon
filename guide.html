下载整合包

安装步骤

  1. 下载整合包压缩文件（.zip）
  2. 打开 PCL2/HMCL 启动器
  3. 点击「版本管理」或「版本列表」
  4. 选择「导入整合包」
  5. 选择下载的压缩文件，等待导入完成

常见问题

游戏启动后闪退怎么办？

1. 检查是否分配了足够的内存（建议 4-8GB）
2. 确保 Java 版本正确（需要 Java 21+）
3. 尝试更新显卡驱动
4. 关闭杀毒软件后重试

连接服务器超时怎么办？

1. 检查网络连接是否正常
2. 确认服务器地址输入正确
3. 尝试使用加速器
4. 检查防火墙是否阻止了游戏

如何获得初始宝可梦？

进入服务器后，在野外行走一段时间，会触发宝可梦博士的对话，届时可以选择你的初始宝可梦（御三家）。

怎么设置皮肤？

1. 注册 LittleSkin 账户
2. 在皮肤站上传你喜欢的皮肤
3. 启动器中使用外置登录（LittleSkin）
4. 进入游戏即可显示皮肤

遇到问题如何求助？

1. 查看服务器公告和帮助文档
2. 在游戏内使用 /help 命令
3. 联系在线管理员
4. 加入我们的 QQ 群：123456789
